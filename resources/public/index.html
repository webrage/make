Hello world

3 September 2018

I have been wanting to play with Cryogen for a while now. It is a clojure based helper for creating static web sites.

Having had a quick look at it I have decided to go with the lotus theme that you can see now. This will be the place where I post about the interesting things I create at my local makerspace (HackLand).

setup issues

cryogen

With any new process there is stuff to learn and figure out. Being familiar with clojure I was quickly able to bring the getting started website to life at localhost.

lotus theme

The fun began when I decided to go with the lotus theme and the website wouldn't render in the browser. This theme uses SASS for its style sheets so I had to install the SASS command-line app and update the config file. I was happy to install this as it looks interesting. Still no luck. Turns out the latest version of the SASS command-line app doesn't work the same as previously, leaving cryogen throwing an error when called. My current work-around is to manually call the SASS executable to parse the scss files creating a css file in the same directory. This is only needed after a scss is updated, which shouldn't be too often.

github

The next challenge was to store the website on github. Not hard, just that I hadn't done it before. Turns out Atom, the IDE I use makes the process pretty easy.
